LiveSplit.VTS
=====================

LiveSplit.VTS is a [LiveSplit](http://livesplit.org/) component that allows for connecting to Vtube Studio and execute actions based on splits.

Requirements
------------

  * LiveSplit 1.5+
  * .NET Framework 4.7.2
